# Memory Bank - Docidv iframe Test Project

## Project Overview

This is an iframe testing application for the Docidv service. It allows users to dynamically change the iframe source URL through a side panel interface.

## Key Features Implemented

1. **Side Panel (Left)**: 300px wide dark-themed panel (#34495e background)

   - URL input field pre-filled with default URL
   - "Apply URL" button
   - Keyboard support (Enter key to apply)

2. **Dynamic URL Management**:

   - Changes iframe src attribute
   - Updates header text to show current URL being tested
   - Shows loading indicator during transitions

3. **Default URL**: `https://go.dev7.idnow.de/docidv?identToken=DV7-RHULQ`

## Technical Implementation Details

### Layout

- Flexbox layout with `.main-content` container
- Side panel: fixed 300px width
- Content area: flex: 1 (takes remaining space)
- iframe container: 100% width, 900px height

### JavaScript Functionality

- Event listener on "Apply URL" button
- Enter key support in input field
- Updates both iframe.src and header text (#currentUrl element)
- Loading indicator management

### iframe Permissions

```html
allow="camera; microphone; geolocation; fullscreen; clipboard-read;
clipboard-write" sandbox="allow-same-origin allow-scripts allow-forms
allow-popups allow-modals"
```
